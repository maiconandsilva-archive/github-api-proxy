## Getting Started

### Installation

1. Install NPM packages
   ```sh
   yarn install
   ```
2. Copy the file `config.js.sample` to `config.js`.
    ```sh
    cp config.js.sample config.js
    ```

### Usage

## Development

```sh
DEBUG=request yarn dev
```

## Production

```sh
PORT=80 yarn start
```
